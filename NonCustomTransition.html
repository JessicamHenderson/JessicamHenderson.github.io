A library is a packaged code that developers can use in projects and helps prevent reinventing the wheel. It helps cut down on the number of copy-pasting. A library ought to be one file, maintained seperately. JQuery is a library, and so is Javascript. When you create your own, it becoms customized, which is something we're trying to avoid in these examples.

When we talk about designing, defining, and triggering in coding terms, designing is the idea of what you want. I want a square to become a circle when someone hovers over it. Defining is when you use the code to create that transition, and triggering is when something happens to make that transition happen.
